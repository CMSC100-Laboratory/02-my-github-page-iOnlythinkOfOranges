**Steps on how to create GitHub page:**

1. In your repository, go to settings.
2. Go to "Code and automation" section, then click Pages.
3. Under "Build and deployment", select Deploy from a branch as "Source".
4. Under "Branch", select a publishing source, then save.
5. Click visit site located at the top of the page.
